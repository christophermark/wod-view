# What's new — store release notes

The "What's New" text for each release, written **before** the version bump
so the tagged commit carries the notes that shipped. Newest first. Used for:

- the GitHub release body,
- App Store Connect → "What's New in This Version" (paste when submitting),
- Google Play → release notes (paste when promoting to production).

Plain, athletic-utilitarian voice, matching the store listing. Never include
anything from the personal dataset (the AGENTS.md privacy rule applies here
too).
